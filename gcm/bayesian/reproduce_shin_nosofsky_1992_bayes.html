Reproduction of model-based analyses by Nosofsky & Shin (1992)

Frederik Aust

To validate this implementation of the Generalized Context Model (GCM) for recognition data, I reproduced small parts of the model-based analyses reported in Nosofsky & Shin (1992). The original MDS solutions and response data were provided by Shin & Nosofsky (1992; s. note on data).

Experiment 1

trials <- 3 * 50

tests <- c()
data <- c()
for(i in 1:3) {
  tests <- rbind(tests, read.csv2(paste0("../data/shin_nosofsky_1992_cat", i, ".csv")))
  data <- rbind(data, read.csv2(paste0("../data/shin_nosofsky_1992_responses_cat", i, ".csv")))
}
data$response <- round(data$Observed * trials)


Summary fits

recognition_data <- list(
  y = data$response
  , tests = as.matrix(tests[, -1])
  , memory = as.matrix(subset(tests, Exemplar %in% paste0("O", 1:6))[, -1])
  , ntrials = rep(trials, nrow(tests))
  , p = 1 # Shape of relationship between similarity and psychological distance
  , rho = 2 # Power of the Minkowski distance
)

poi <- c("c", "w", "k", "pred_y")

# Sample
gcm_samples <- run.jags(
  model = "GCM_agg_recognition.txt"
  , monitor = poi
  , inits = init_values
  , data = recognition_data
  , n.chains = 3
  , sample = 5e4
  , burnin = 100
  , thin = 5
  , method = "rjparallel"
)
